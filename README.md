# Spotify Song Analyzer

Spotify Song Analyzer is a Python Flask web application that analyzes songs using audio features like energy, danceability, popularity, and tempo. The app recommends similar songs, compares tracks, and displays music analytics in a clean interactive interface.

## Features

- Search for songs
- View song statistics
- Get song recommendations
- Compare songs side-by-side
- Mood classification system
- Interactive visualizations
- Responsive modern UI

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Git/GitHub

## How It Works

The recommendation system compares songs based on:
- Energy
- Danceability
- Popularity
- Tempo

Songs with the most similar feature values are recommended to the user.
